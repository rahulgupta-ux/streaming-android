# Streaming Android App

Android client for a Netflix-style streaming platform using XML UI and ExoPlayer.

## Tech Stack
- Kotlin
- XML Layouts
- ExoPlayer (HLS streaming)
- Retrofit + OkHttp
- DataStore (for JWT token)
- MVVM Architecture

## Features
- Login / Signup
- Home screen with video list
- Video detail screen
- ExoPlayer HLS streaming
- Continue watching
- Search
